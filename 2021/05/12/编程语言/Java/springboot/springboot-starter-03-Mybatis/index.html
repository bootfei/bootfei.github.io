springboot-starter-03-Mybatis

Posted on 2021-05-12

导入三个依赖

  • mybatis 与 Spring Boot 整合依赖，mysql 驱动依赖， Druid 数据源依赖。

  • <!--mybatis 与 spring boot 整合依赖-->
<dependency>
    <groupId>org.mybatis.spring.boot</groupId>
  <artifactId>mybatis-spring-boot-starter</artifactId>
</dependency>

<!--mysql 驱动-->
<dependency>
    <groupId>mysql</groupId>
  <artifactId>mysql-connector-java</artifactId>
</dependency>

<!-- druid 驱动，数据源 -->
<dependency>
    <groupId>com.alibaba</groupId>
  <artifactId>druid</artifactId>
  <version>1.1.12</version>
</dependency>

定义 Dao 接口

  • Com.abc.dao目录

  • //Dao 接口上要添加@Mapper 注解。
public interface IstudentDao{
    void insertStudent(Student s);
}

  • 注册资源目录

    • 在 pom 文件中将 dao 目录注册为资源目录。

    • <!-- dao 目录注册为资源目录 -->
<build>
  <resources>
    <resource>
        <directory>src/main/java</directory>
      <includes>
        <include>**/*.xml</include>
      </includes>
    </resource>
  </resources>
</build>

    • 注册实体类别名

      • mybatis.type-aliases-packages=com.abc.beans
