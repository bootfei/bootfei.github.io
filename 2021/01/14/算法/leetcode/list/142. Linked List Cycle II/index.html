Floyd’s Tortoise and Hare

The nodes in the cycle have been labelled from 0 to C−1, where C is the length of the cycle.

The noncyclic nodes have been labelled from −F to -1, where F is the number of nodes outside of the cycle.

​ C-1

​ <-node <-node ….

node node node … node node

​ ->node-> node->

-F -F+1 0

phase1:

  • After F iterations, Tortoise arrives at node 0 and Hare points to node h then

    F ≡ h( mod C)
    This is because Hare traverses 2F nodes over the course of F iterations, exactly F of which are in the cycle.

  • After C-h more iterations, Tortoise arrives at node h， and Hare also points to node h, why?
    remember that hare traverses 2(C-h) from its starting position of h, then h + 2(C-h) = 2C-h ≡ C-h(mod C)

Therefore, given that the list is cyclic, hare and tortoise will eventually both point to the same node, known hence as the intersection.

phase2:

  • Given that phase 1 finds an intersection, phase 2 proceeds to find the node that is the entrance to the cycle. To do so, we initialize two more pointers: ptr1, which points to the head of the list, and ptr2, which points to the intersection.

  • Then, we advance each of them by 1 until they meet; the node where they meet is the entrance to the cycle, so we return it.

    如何证明呢？

    经过一定的iteration以后, 乌龟和兔子都到达h, 那么兔子的路程 = 2 * 乌龟的路程

    2(F+a)=F+nC+a ==> F + a = nC 非常重要的结论

    然后，新的ptr1从 node -F出发走到node 0，期间经过了F个iteration; 那么ptr 2根据上面的结论，抵达的位置是 a + iteration = a + F = a + (nc -a ) = nc ≡ 0 ( mod c), 结论得证。
