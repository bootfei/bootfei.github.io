搭建hexo博客 and 部署在阿里云上

###使用命令写一篇文章

hexo new “title test:标题test”

###使用以下步骤删除一篇文章

  1. Delete the post under source/_post folder
  2. Run hexo clean to delete the database (db.json) and assets folder
  3. Run hexo generateto generate the new blog without your deleted post
  4. Run hexo deploy to deploy your blog

refer: https://stackoverflow.com/questions/27894210/how-do-i-delete-a-post-in-hexo

使用合适的编译器编辑文章

  1. 使用sublime Text 3的方式
    a. 在线mk编辑与预览https://stackedit.io/editor 从而可以分屏查看markdown的预览
     refer: https://zhihu.ocm/question/27384681
    b. 安装Livereload插件。
     hexo s -g在chrome上预览, 打开chorme的livereload插件。
