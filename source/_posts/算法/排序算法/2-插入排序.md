思路：在洗好的牌中，插入一张牌插入到其中合适的位置；索引左边的数组都是已经有序的



```java

	public void selectSort(int[] arr){
		int n = arr.length;
		for(int i=1; i<n; i++){

			//将a[i]插入到a[i-1],a[i-2]....,a[0]中
			for(int j=i; j>0; j--){
				if(arr[j]<arr[j-1]){
					swap(arr, j-1, j);
				}else{
					break;
				}
			}

		}
	}


	public void swap(int[] arr, int i, int j){
		int tmp = arr[i];
		arr[i] = arr[j];
		arr[j] = tmp;
	}

```



优点：对于数量较少的数组，这个算法是最快的
