思路：1.从当前数组中遍历，找到数组最小的数 2.最小的数与数组的第一个元素交换位置  3. 在剩下的数组中重复步骤1



```
public void selectSort(int[] arr){
	for(int i=0; i<n; i++){
		int min = i;
		for(int j=i+1; j<n; j++){
			if(arr[min]>arr[j]){
				min = j;
			}
		}
		swap(arr, i, min);
	}
}
```



缺点：

1.  为了找出当前数组的最小值而扫描一次数组，并不能为下一次扫描数组提供更多信息
